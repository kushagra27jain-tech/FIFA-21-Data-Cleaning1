# ⚽ FIFA 21 Data Cleaning Challenge

## 📌 Project Overview

This project focuses on cleaning and preprocessing the FIFA 21 player dataset using **Python**, **Pandas**, and **NumPy**.

The raw dataset contained missing values, inconsistent formatting, incorrect data types, duplicate records, and unnecessary columns. The objective was to transform the dataset into a clean, analysis-ready format following industry-standard data cleaning practices.

---

## 🎯 Objectives

- Clean messy real-world data
- Handle missing values
- Convert incorrect data types
- Remove duplicate records
- Standardize text values
- Prepare the dataset for Exploratory Data Analysis (EDA) and Machine Learning
